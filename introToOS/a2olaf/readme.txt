Copies the CatchingAHorse folder and StudioPoseOlaf.obj file to the current user's H: drive

# *** How to use ***
---------------------------------------
Syntax
	generateBuild [substring_to_find]

substring_to_find defaults to "v "

"v " counts vertices
"vt " counts texture coordinates
"vn " counts vertex normals
"vp " counts parameter space vertices
"f " counts face elements
"l " counts line elements

any other value for substring_to_find will still be searched

Make sure to run the cmd script within the folder containing the CatchingAHorse folder and the StudioPoseOlaf.obj file
